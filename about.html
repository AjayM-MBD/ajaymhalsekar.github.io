About

I'm Ajay — an automotive BEV engineer with hands-on experience in product development, control algorithm design, and embedded software for braking and electric drive systems. This site is a collection of technical tutorials, case studies, and code examples focused on:

  • Electronic Parking Brake (EPB) systems
  • EV drive units and motor control
  • ADAS algorithm analysis
  • Model-based design (Simulink) and Embedded C

This site is small, focused, and free — built to help engineers and students learn practical techniques without fluff.
